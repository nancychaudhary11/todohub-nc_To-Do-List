# 📝 TodoHub

TodoHub is a simple and interactive **To-Do List** web application built using **HTML, CSS, and JavaScript**. It allows users to manage their daily tasks through a clean and easy-to-use interface.

## 🌐 Live Demo

🔗 https://todohub-nc.netlify.app/

## ✨ Features

- ➕ Add new tasks
- ✅ Mark tasks as completed
- 🗑️ Delete tasks
- 🎨 Clean and simple user interface
- ⚡ Fast and lightweight
- 📦 Progressive Web App (PWA) setup using `manifest.json`

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
